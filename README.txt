Description:
- Important Note: My code uses ResourceManager to load resources such as images etc. This is supported by Java version
  8.065 but might be dericated in future releases
- Note: Large image files captured by a DSLR might take time to load
- To clear the filter: click on the 5th star twice (one for chosing the 5th filter and second for clearing all)
- To clear the ranking: click on the 5th ranking star twice
- Click on the folder icon to add images
- Title bar includes options for list view as well as grid view
- State will be saved on exit
- Saved state will be loaded on startup
- Filters are at the bottom
- Preferred Environment: Mac

Enjoy!
